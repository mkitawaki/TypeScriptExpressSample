#TypeScript Express Sample

##Install

Install TypeScript

```
npm install -g typescript
```

##Getting Started

```
git clone https://github.com/mkitawaki/TypeScriptExpressSample.git

```

```
npm install
```

Compile TypeScript

```
tsc --module commonJs app.ts routes/index.ts routes/user.ts
```

##Run

```
node app.js
```

Web Browser Access

```
http://localhost:3000
```
